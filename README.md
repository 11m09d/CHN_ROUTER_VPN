# CHN_Router_VPN

## 架构

Client <—VPN—> China VPS <—Shadowsocks—>America VPS

## 优点

1. 兼容性好，VPN干扰严重但大部分设备都支持，SS稳定但移动设备体验不够好。
2. 国内VPS中转，可选择出口线路优质的VPS，提高出口速度。
3. 可在国内VPS根据国内外IP地址分流，提高访问速度，节省流量。
